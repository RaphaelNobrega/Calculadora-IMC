
# Calculadora IMC

Este projeto foi desenvolvido com foco no aprendizado de manipulação de elementos e utilização de condicionais em JavaScript.


## Referência

## Stack utilizada

**Front-end:** HTML, CSS, JavaScript, Bootstrap
